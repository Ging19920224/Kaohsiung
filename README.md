# 高雄旅遊資訊

## DEMO
  <https://ging19920224.github.io/Kaohsiung/>

## 功能
  * 根據使用者下拉選單，顯示出景點資訊

## 開發技術
  * JavaScript XMLHttpRequest 串接 API
  * API 來源：[高雄旅遊網-景點資料](https://data.kcg.gov.tw/dataset/attractions-information)
  * JavaScript ES6
